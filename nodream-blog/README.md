## nodream-blog

### 技术栈
Vue2 + Vue-Router + Django + Axios + Element-Ui

### 文件结构
 - assets：存放静态资源
 - common：存放公用js
 - components：存放公用组件
 - network：网络请求文件
 - router：路由文件
 - store：全局状态管理
 - views：视图文件
